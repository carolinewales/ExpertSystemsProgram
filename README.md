# Expert System Program - Application Qualification
CSC371 - Caroline Wales

This expert system replicates a rule-based engine. It asks a user about their qualifications, coursework, skills, and experience. It then uses this information to figure out which jobs they are qualified for, and which they are not. If a user is unqualified for a job, the reason(s) why are given.

How to Run:

Download ExpertSystemProgram.java
Open a new terminal
Compile: javac ExpertSystemProgram.java
Run: java ExpertSystemProgram
